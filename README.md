
# TUTORKU Frontend

Frontend aplikasi edukasi TUTORKU yang dibangun dengan React, TypeScript, Vite, dan Tailwind CSS. Aplikasi ini mencakup alur untuk siswa, tutor, serta admin platform.

## Ringkasan proyek

Proyek ini merupakan antarmuka web modern untuk platform belajar online dengan fitur seperti:

- pendaftaran dan login pengguna
- pencarian tutor dan detail tutor
- pemesanan kelas dan riwayat belajar
- upload video materi
- dashboard admin dan platform admin
- chat, notifikasi, serta integrasi live class

## Teknologi utama

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- MUI / Radix UI / shadcn-style components
- i18next untuk dukungan bahasa

## Prasyarat

Pastikan perangkat sudah terpasang:

- Node.js 18+
- pnpm

## Instalasi

```bash
pnpm install
```

## Menjalankan aplikasi

```bash
pnpm dev
```

Aplikasi akan berjalan di browser sesuai URL lokal yang ditampilkan oleh Vite.

## Build produksi

```bash
pnpm build
```

## Struktur folder penting

- src/app: halaman dan komponen aplikasi
- src/app/auth: halaman autentikasi
- src/app/mahasiswa: fitur untuk siswa
- src/app/TUTORKU: fitur tutor dan modul inti
- src/app/admin: panel admin
- public: aset statis

## Catatan

Beberapa file sertifikat lokal tersedia di folder certs untuk kebutuhan pengembangan HTTPS.
